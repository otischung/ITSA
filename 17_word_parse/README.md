# 題目17. 英文斷詞

## 問題描述

斷詞在自然語言的研究上是個很重要的步驟，主要就是將關鍵字從句子中斷出，英文的斷詞較為簡單，就根據句子中的空格將英文字隔開。

**輸入格式**
輸入一句英文敘述句。 字元數≤1000。

**輸出格式**
將輸入的句子進行斷詞，將斷出的關鍵字依照句子中的出現排序列印出。全部轉成小寫，列印出的關鍵字不得重複，關鍵字間以一個空格隔開，最後一個關鍵字後面進行換行。例如輸入 How do you do ，則輸出 how do you 。

**範例**

| 輸入範例      | 輸出範例   |
| ------------- | ---------- |
| How do you do | how do you |
